</div>

Для генерации случайного цвета используй функцию getRandomHexColor.

function getRandomHexColor() {

return `#${Math.floor(Math.random() * 16777215)

.toString(16)

.padStart(6, 0)}`;
}

function getRandomHexColor() {

return `#${Math.floor(Math.random() * 16777215)

.padStart(6, 0)}`;
